# 墨屿 · 文学疗愈空间

在诗词和文学中，安放你的迷茫、焦虑与疲惫。

## 关于

墨屿是一个安静的文学探索空间，收录了跨越千年的经典诗词，按情绪分类（迷茫时、焦虑时、孤独时、需勇气、疲惫时）。每首诗词配有白话译文和一段温暖的微赏析，让古典文学在现代生活中继续治愈人心。

## 功能

- **每日推荐** — 每天一首不同的诗词
- **按情绪探索** — 根据你当下的心境选择对应分类
- **随机邂逅** — 随机遇到一首诗（快捷键 `R`）
- **收藏** — 收藏你喜欢的诗词（快捷键 `F`）
- **便签** — 写下你的读诗感悟，保存在浏览器中

## 技术

纯前端，HTML + CSS + JavaScript，零依赖。数据储存在浏览器的 localStorage 中。可直接部署到 GitHub Pages。

## 本地使用

直接打开 `index.html` 即可，无需服务器。

## 部署到 GitHub Pages

1. 在 GitHub 上创建一个新仓库
2. 将本项目文件推送到仓库的 `main` 分支
3. 在仓库 Settings > Pages 中，选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/ (root)`
4. 几分钟后即可通过 `https://<你的用户名>.github.io/<仓库名>/` 访问
